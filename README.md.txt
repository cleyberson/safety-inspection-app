# Safety Inspection App

Aplicação desenvolvida em Python para digitalização de inspeções de segurança do trabalho.

## Objetivo

Substituir formulários impressos utilizados em inspeções de segurança por um sistema digital capaz de registrar dados, armazenar informações e gerar indicadores de segurança.

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Matplotlib
- Pillow

## Funcionalidades

- Registro digital de inspeções de segurança
- Upload de imagens das não conformidades
- Armazenamento de dados das inspeções
- Geração de gráficos para análise de indicadores

## Como executar o projeto

1. Instalar as bibliotecas necessárias

pip install -r requirements.txt

2. Executar a aplicação

streamlit run app.py

## Contexto do projeto

Este projeto foi desenvolvido como iniciativa de digitalização de processos de inspeção de segurança em ambiente industrial, substituindo formulários em papel por uma solução digital utilizando Python.
